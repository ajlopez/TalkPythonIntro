## Running Python hello.py ```python print('Hello, world') ``` Running hello.py ```python python hello.py ```
## Unix/Linux Add `shebang` at the beginning of the file ``` #!/usr/bin/python ``` If you don't know where is `python` ``` #!/usr/bin/env python ``` Make `hello.py` executable ``` chmod a+x hello.py ```

Installing pip

See installing pip

In my Windows 8 machine:

  • Install https://pypi.python.org/pypi/distribute/
  • Download distribute_setup.py from http://python-distribute.org/
  • Execute python distribute_setup.py
  • Execute curl -O https://raw.github.com/pypa/pip/master/contrib/get-pip.py
  • Execute python get-pip.py
  • Add <your python>\Scripts to your path
